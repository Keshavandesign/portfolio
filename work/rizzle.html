01 The problem

Four teams.
Four button styles.
Zero alignment.

Rizzle's product moved fast. Every squad pulled from its own Figma library or copied components off existing screens. After two years: four distinct button styles, three spacing systems, values that lived only in engineers' memory.

I joined to fix it. Tokens first. Components second. Documentation last — because docs nobody reads are just debt with better formatting.

1
Audit the existing product
Two weeks of screenshots, tagging every unique spacing value, color, and component variant. Found 23 different shades of purple being used across the product.
2
Define the token architecture
Primitive tokens first (raw values), then semantic tokens (what they mean: surface, interactive, error). Engineers adopted the semantic layer directly into CSS custom properties.
3
Build molecules from atoms
Every state documented explicitly — idle, hover, active, disabled, error. If it wasn't in the library, it didn't exist.
4
Test inside real product flows
Embedded the library into three ongoing builds while still in progress. Gaps showed up fast. Better then than after 40 engineers have adopted draft one.
5
Ship the docs as a product
Built inside Rizzle's own stack, not Notion. If designers had to leave their environment to read the rules, they wouldn't.

05 Foundation · Color

Color system

Purple is the brand.
Semantics are the system.

Full tonal scale — 50 to 900. Every surface, from subtle tints to vibrant CTAs, pulls from the same source.

Semantic tokens sit on top: color.interactive.idle, color.error.idle. State behavior is a token decision, not a one-off. When Rizzle refreshed their brand, we swapped values under the tokens and the whole product updated in a day.

11 What I took away

A design system is a political document as much as a technical one.

01
Token adoption requires trust, not enforcement. Engineers used the system because it made their work easier. The docs had to answer questions they were already asking — not questions I assumed they had.
02
Name for intent, not appearance. color.interactive.idle ages better than purple-500. When Rizzle refreshed their brand, we swapped values under the tokens and the whole product updated in a day.
03
Ship an incomplete system sooner. Three squads on a 60% complete library gave us real feedback in two weeks that six more months of spec work never would have surfaced. Zero major revisions post-handoff. I'll take that.
